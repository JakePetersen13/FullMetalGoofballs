# FULL METAL GOOFBALLS
## Setup
1. Download the .zip file
2. Extract all files from the zip
3. Open the CS4700_Project.exe file in the extracted file
## Features
* Chaotic Gameplay
  * Over the top ragdoll physics
  * Explosions everywhere
  * Random weaponry
* Absurd Objectives
  * Ex: Sabotage the enemy grill during the BBQ while defending your own
* Fun over Realism
  * Allows more unconventional additions to gameplay, artstyle, UI, etc.
